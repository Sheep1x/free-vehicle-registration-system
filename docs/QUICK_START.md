# 智票通 - 快速开始指南

## 项目简介

智票通是一款专业的车辆通行费票据识别微信小程序，通过AI技术自动提取票据信息，提高报销效率。

---

## 技术架构

```
前端：Taro + React + TypeScript + Tailwind CSS
数据库：Supabase
AI能力：文心一言多模态大模型
```

---

## 核心功能

### 1️⃣ 票据识别
- 拍照或上传票据图片
- AI自动识别关键信息
- 支持手动编辑修正

### 2️⃣ 历史记录
- 云端存储所有记录
- 车牌号搜索筛选
- 批量删除管理

### 3️⃣ 个人中心
- 使用帮助说明
- 关于应用信息
- 导出功能（规划中）

## 使用流程

```
1. 打开小程序
   ↓
2. 选择/拍摄票据图片
   ↓
3. 点击"开始识别"
   ↓
4. AI自动提取信息（3-5秒）
   ↓
5. 确认或编辑结果
   ↓
6. 保存到历史记录
   ↓
7. 随时查看和管理
```

## 数据存储

### 本地存储
- 无（所有数据存储在云端）

### 云端存储
- **数据库**：Supabase PostgreSQL
- **表名**：toll_records
- **存储内容**：所有识别记录
- **数据安全**：加密传输，安全可靠

## 常见问题

### Q: 识别需要多长时间？
A: 通常需要3-5秒，取决于网络状况和图片大小。

### Q: 识别准确率如何？
A: 采用文心一言多模态大模型，准确率较高。如有误差可手动修正。

### Q: 数据会丢失吗？
A: 所有数据存储在云端Supabase数据库，不会丢失。

### Q: 支持哪些图片格式？
A: 支持JPG、JPEG、PNG、BMP等常见格式。

### Q: 可以批量识别吗？
A: 当前版本暂不支持，规划在后续版本中实现。
