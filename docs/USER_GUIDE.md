# 智票通使用指南

## 功能介绍

智票通是一款专业的车辆通行费票据识别小程序，帮助您快速提取票据信息，提高报销效率。

---

## 主要功能

### 1. 票据识别

**操作步骤：**

1. 打开小程序，进入"识别"页面
2. 点击"选择图片"按钮
3. 选择拍照或从相册选择票据图片
4. 确保票据图片清晰，文字可辨
5. 点击"开始识别"按钮
6. 等待AI识别完成（通常需要3-5秒）

**识别内容：**
- 车牌号（如：蓝 鲁P233CV）
- 车型（如：一型 客车）
- 轴数（如：2轴）
- 吨位（如：1.26吨）
- 入口信息（如：河北清河站）
- 通行时间（如：2025-12-06 13:25:05）
- 金额（如：79.00元）

**注意事项：**
- 确保光线充足，避免反光
- 票据文字清晰可见
- 尽量保持票据平整
- 支持JPG、PNG、BMP等常见格式

---

### 2. 识别结果确认

识别完成后，系统会跳转到结果页面：

1. **查看识别结果**：所有提取的信息会显示在表单中
2. **编辑修正**：如果识别有误，可以手动修改任何字段
3. **保存记录**：确认无误后，点击"保存记录"按钮
4. **返回重试**：如果需要重新识别，点击"返回"按钮

---

### 3. 历史记录管理

在"记录"页面可以查看和管理所有识别记录：

**查看记录：**
- 显示所有历史识别记录
- 包含票据缩略图、车牌号、金额等关键信息
- 按时间倒序排列，最新记录在最前

**搜索功能：**
1. 在顶部搜索框输入车牌号
2. 点击"搜索"按钮
3. 系统会筛选出匹配的记录

**批量删除：**
1. 点击右上角"管理"按钮
2. 选择要删除的记录（可多选）
3. 点击"删除"按钮
4. 确认删除操作

**快捷操作：**
- 点击"全选"可选中所有记录
- 点击"取消全选"可取消所有选择
- 点击"完成"退出管理模式

## 常见问题

### Q1: 识别不准确怎么办？
**A:** 
- 确保票据图片清晰，光线充足
- 避免拍摄时产生阴影或反光
- 识别后可以手动编辑修正结果

### Q2: 支持哪些票据类型？
**A:** 
- 目前主要支持车辆通行费电子票据
- 支持各省份高速公路通行费票据

### Q3: 识别记录保存在哪里？
**A:** 
- 所有记录保存在云端数据库
- 可以随时查看和管理
- 数据安全可靠

### Q4: 可以删除记录吗？
**A:** 
- 可以删除单条或批量删除记录
- 删除后无法恢复，请谨慎操作

### Q5: 识别需要多长时间？
**A:** 
- 通常需要3-5秒
- 具体时间取决于网络状况和图片大小

## 使用技巧

1. **拍照技巧**
   - 保持手机稳定，避免模糊
   - 票据占据画面大部分区域
   - 避免背景杂乱

2. **提高识别准确率**
   - 使用原始票据图片，避免截图
   - 确保票据文字清晰可见
   - 光线均匀，避免过曝或过暗

3. **高效管理**
   - 及时保存识别结果
   - 定期整理历史记录
   - 使用搜索功能快速查找
