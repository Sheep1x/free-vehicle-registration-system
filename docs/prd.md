# 免费车登记小程序+后台管理系统需求文档

## 1. 小程序概述
### 1.1 小程序名称
免费车登记

### 1.2 小程序描述
专门用于登记免费通行车辆信息的微信小程序，通过拍照或上传图片自动提取车辆关键信息，提高免费车辆管理和登记效率。

## 2. 小程序核心功能
### 2.1 拍照上传功能
- 支持调用手机摄像头拍照
- 支持从相册选择图片上传\n- 图片预处理（裁剪、增强清晰度）
- 首页悬浮拍照按钮，直接调用摄像头拍照后识别

### 2.2 OCR识别功能
- 车牌号码识别：自动提取车辆中的车牌号信息
- 车型识别：识别车辆类型
- 时间信息提取：提取通行时间
- 车辆信息：车型、轴数等

### 2.3 识别结果展示
- 清晰展示提取的所有关键信息
- 支持结果编辑和修正
- 免费原因选择：紧急车、军警车、应急车、旅游包车（下拉选择框）
- 收费员选择（下拉框）
- 监控员选择（下拉框）
- 保存记录按钮左侧设置重新识别按钮
- 左上角设置返回按钮（<图标）

### 2.4 历史记录
- 保存历次识别结果
- 支持按时间、车牌号筛选
- 批量删除功能
- 增加交互式操作（可点击查看详情、编辑等）

## 3. HTML后台管理系统
### 3.1 用户管理模块
#### 3.1.1 收费员管理
- 收费员信息录入（姓名、工号、联系方式等）
- 收费站级别分组管理
- 收费班组级别分组管理
- 班组班次分配

#### 3.1.2 监控员管理
- 监控员信息录入（姓名、工号、联系方式等）
- 对应收费站分配
- 班组班次分配

### 3.2 组织架构管理
#### 3.2.1 收费站管理
- 收费站信息录入和维护
- 收费站下辖收费班组管理
- 班组间人员调配

#### 3.2.2 班组管理
- 白班：7:30-15:30
- 中班：15:30-23:30
- 夜班：23:30-7:30
- 支持自定义班次时间和名称
- 班组成员排班管理

### 3.3 识别记录管理
#### 3.3.1 记录查询
- 按时间范围、车牌号、收费员、监控员等条件筛选
- 支持批量操作
- 实时同步小程序识别数据

#### 3.3.2 数据导出
- 支持导出为Excel格式
- 支持导出为PDF格式
- 支持自定义导出字段
- 支持定时批量导出

## 4. 参考素材
### 4.1 标记图片
使用用户提供的图片蓝牌标记.png作为OCR识别的参考模板，该图片已清晰标记了需要识别的各个区域位置。

## 5. 设计风格
### 5.1 小程序设计风格
- **配色方案**：主色调采用蓝色系（#1890FF），辅以白色背景和灰色文字，体现专业可靠的科技感
- **视觉细节**：界面采用卡片式布局，圆角设计增加亲和力，识别结果使用表格形式清晰展示各项数据
- **交互体验**：拍照界面采用悬浮按钮设计，识别结果页面支持滑动刷新，整体操作流畅便捷

### 5.2 后台管理系统设计风格
- **配色方案**：深蓝色主题色配合浅灰色背景，营造专业稳重的工作氛围
- **布局方式**：采用经典的顶部导航+左侧菜单+主内容区的后台布局结构
- **视觉细节**：表格采用斑马纹设计增强可读性，按钮使用圆角矩形，整体保持简洁统一的视觉风格
